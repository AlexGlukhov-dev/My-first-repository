git clone path to repository - команда создающая копию гита на вашем девайсе
git status - cтатус текущего гит репозитория
git add path to file
git commit -am 'added new file index.html' - зафиксировать изменения и дать им коментарий
git push - отправить все закомиченные изменения на сервер репозитория
git pull - забрать свежие обновления
